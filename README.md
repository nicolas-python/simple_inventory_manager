# Simple Inventory Manager

Ein Inventory Manager zum Hinzufügen, Anzeigen und Löschen von Produkten.

## Funktionen 

- Hinzufügen von Produkten
- Anzeigen von Produkten
- Löschen von Produkten
- Produkte werden in einer Datei gespeichert

## Nutzung

Programm starten mit : 

python main.py

Danach im Menü auswählen.
Je nach auswahl fortfahren.
Beim Beenden des Programms mit Option 4
werden die Listen automatisch gespeichert.
